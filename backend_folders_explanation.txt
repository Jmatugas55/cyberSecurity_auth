backend/ folder: main folder sa python FastAPI.
app/ folder : all the backend logic is here.
pycache/: This folder is automatically generated by Python.
routers/ : This folder contains the API route definitions.
	Routes define the endpoints that the frontend can call.
example:
	POST /login
	POST /register
	POST /change-password
Files inside routers/ folder:
	1. auth.py : Handles authentication-related endpoints.
	2. login_attempt.py: Tracks login attempts.
	3. password.py : Handles password-related actions.

Files inside app/ folder:

	1. crud.py: Handles database operations.
	2. database.py : This file manages the database connection.
	3. main.py : This is the entry point of the backend server.
	4. models.py : This defines the database table structure.
	5. schemas.py : This defines the data validation models.
	6. security.py : Handles security functions (e.g password hashsing, jwt token creation, and et.).
	7. utils.py : Utility/helper functions.Store reusable functions used across the system.
